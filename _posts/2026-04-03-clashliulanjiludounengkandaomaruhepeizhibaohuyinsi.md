<h3>实用小工具常见问题 (FAQ)</h3>
<p>在使用过程中，您可能会遇到一些常见问题。这里了几个高频疑问并提供解答。</p>
<ul>
<li><strong>1. 如何确认代理是否生效？</strong><br />
    最简单的方法是访问一个可以显示您 IP 地址的网站。您也可以使用命令行工具进行检查（需要将端口号 `7890` 替节点clash订阅换为您客户端的实际端口）：<br />
    <code>curl -x http://127.0.0.1:7890 https://ifconfig.me</code><br />
    如果返回的 IP 地址您的真实 IP 不同，则说明代理已生效。常用的在线检测网址有：`ip.sb` 或 `ipinfo.io`。
  </li>
<li><strong>2. 局（Global）、规则（Rule）和直连（Direct）模式有什么区别？</strong><br />
    - <strong>局模式：</strong> 所有网络流量都通过您选择的代理节点。<br />
    - <strong>规则模式：</strong> 根据预设的规则列表，自动判断哪些流量走代理，哪些直连（例如，国内网站直连，国外网站走代理）。这是最推荐的日常使用模式。<br />
    - <strong>直连模式：</strong> 所有流量都不走代理，等同于关闭了代理功能。
  </li>
<li><strong>3. 代理节点为什么我的速度还是很慢？</strong><br />
    速度慢可能由多种因素导致：您选择的节点当前负载过高、您的本地网络运营商对出境线路有限制、或者是您所使用的协议（如旧版 <strong>SSR</strong>）特征较为明显。您可以尝试切换不同的<strong>高速线路</strong>或协议（如 <strong>Trojan</strong> 或 V2Ray）来改善。
  </li>
<li><strong>4. 节点服务商到底能看到我的什么信息？</strong><br />
    再次强调，对于 HTTPS 网站，服务商只能看到您请求的域名、IP、时间戳和流量大小。他们无法看到具体内容。对于 HTTP 网站，理论上可以看到所有信息。因此，确保您访问的网站启用了 HTTPS (地址栏有锁形标志) 是一个好习惯。这在一定程度上解答了用户对于“clash浏览记录都能看到吗”的担忧。
  </li>
</ul>
