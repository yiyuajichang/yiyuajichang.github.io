<h3>核心操作：不同平台下 clash如何关闭</h3>

<h4>在 Windows 平台上关闭 Clash</h4>
<p>在 Windows 系统中，Clash for Windows (CFW) 是最常见的客户端。关闭它的步骤如下：</p>
<ol>
<li><strong>第一步：关闭系统代理。</strong> 打开 Clash for Windows 应用主界面，在左侧导航栏找到 “General”（常规）选项卡。</li>
<li><strong>第二步：找到核心开关。</strong> 在 “General” 页面中，您会看到一个名为 “System Proxy”（系统代理）的开关按钮。这个按钮是控制局网络流量是否经过 Clash 的核心。请点击此开关，使其变为灰色关闭状态。</li>
<li><strong>第三步：退出应用程序。</strong> 关闭系统代理后，您可以右键点击任务栏右下角的 Clash 猫咪图标，在弹出的菜单中选择 “Quit” 或 “退出”。这样就彻底完成了关闭操作，系统网络将恢复正常。<em>仅仅点击窗口的关闭按钮 (X) 通常只会将其最小化到系统托盘，而不会关闭代理。</em></li>
</ol>
<h4>在 macOS 平台上关闭 Clash</h4>
<p>对于 macOS 用户，无论是使用 ClashX 还是 Clash for Mac，操作逻辑都非常相似：</p>
<ol>
<li><strong>第一步：通过菜单栏图标操作。</strong> 点击屏幕顶部菜单栏中的 Clash 猫咪图标，会弹出控制菜单。</li>
<li><strong>第二步：取消系统代理。</strong> 在菜单中，找到 “设置为系统代理” (Set as System Proxy) 的选项，确保其前面的对勾（✓）消失。这是最关键的一步，它会清除系统网络设置中的代理配置。</li>
<li><strong>第三步：完退出。</strong> 再次点击菜单栏图标，选择 “退出” (Quit) 选项，即可安地关闭 Clash 客户机场加速端。</li>
</ol>
