# 一元机场官方导航 - 移动端优化版

现代化的机场导航应用，采用 React 19 + TailwindCSS 4 构建。

## 功能特性

### 首页
- 一元机场详细介绍和技术规格
- 性能指标展示（速度、节点数、延迟）
- 支持的协议和解锁服务
- IP 类型和分布信息
- 价格方案对比

### 分类页
- 7 个机场品牌展示（一元、泰山、百变小樱、鳄鱼、樱花、赔钱、木瓜云）
- 按价格、速度、节点数排序
- 完整的对比表
- 功能对比（支持协议、解锁服务）

### 机场详情页
- 单个机场的完整信息
- 技术规格和解锁服务
- 多个价格方案
- 常见问题解答

### 设计特点
- 现代极简科技风格
- 完全响应式设计
- 深蓝色 + 橙色配色
- Poppins 显示字体 + Inter 正文字体
- 流畅的交互动画

## 技术栈

- **前端框架**: React 19
- **样式**: TailwindCSS 4
- **UI 组件**: shadcn/ui
- **路由**: Wouter
- **构建工具**: Vite

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm run dev

# 生产构建
pnpm run build

# 预览构建结果
pnpm run preview
```

## 项目结构

```
app/
├── client/
│   ├── public/          # 静态资源

│   │   ├── pages/       # 页面组件
```

## 机场数据

应用包含 7 个机场品牌的完整数据：

1. **一元机场** - 性价比之王
2. **泰山机场** - 企业级稳定性
3. **百变小樱机场** - 节点丰富多样
4. **鳄鱼机场** - 速度怪兽
5. **樱花机场** - 日本优化专家
6. **赔钱机场** - 价格屠夫
7. **木瓜云机场** - 云端之选

每个机场包含：
- 详细描述和特性
- 技术规格（速度、节点数、延迟）
- 支持的协议
- 解锁的服务
- 多个价格方案

## 自定义

### 修改颜色
编辑 `client/src/index.css` 中的 CSS 变量。

### 添加新机场
编辑 `client/src/lib/airportData.ts`，按照现有格式添加新的机场数据。

### 修改内容
各页面内容位于 `client/src/pages/` 目录中，可直接编辑。

## 部署

项目已针对 Manus 平台优化。点击 UI 中的"Publish"按钮即可部署。
