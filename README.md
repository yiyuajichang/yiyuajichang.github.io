# 一元机场官方导航 - 移动端优化版

现代化的机场导航应用，采用 React 19 + TailwindCSS 4 构建。

## 功能特性

### 首页
- 一元机场详细介绍和技术规格
- 性能指标展示（速度、节点数、延迟）
- 支持的协议和解锁服务
- IP 类型和分布信息
- 价格方案对比

### 分类页
- 7 个机场品牌展示（一元、泰山、百变小樱、鳄鱼、樱花、赔钱、木瓜云）
- 按价格、速度、节点数排序
- 完整的对比表
- 功能对比（支持协议、解锁服务）

### 机场详情页
- 单个机场的完整信息
- 技术规格和解锁服务
- 多个价格方案
- 常见问题解答

## 技术栈

- **前端框架**: React 19
- **样式**: TailwindCSS 4
- **UI 组件**: shadcn/ui
- **路由**: Wouter
- **构建工具**: Vite

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm run dev

# 生产构建
pnpm run build
```
